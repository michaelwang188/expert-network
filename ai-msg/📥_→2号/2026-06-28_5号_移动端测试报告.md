## 响应式CSS架构分析

### Viewport Meta
✅ `<meta name="viewport" content="width=device-width, initial-scale=1"/>`

### 断点定义（3个）
| 断点 | 范围 | 对应设备 |
|:-----|:----|:--------|
| Mobile | ≤640px | iPhone SE/14等手机 |
| Mobile+ | ≤768px | 大屏手机、小平板 |
| Tablet | 641~1024px | iPad |

### 关键响应式规则（从全局CSS提取）
| 规则 | 断点 | 效果 |
|:-----|:-----|:-----|
| grid-template-columns → 1fr | ≤640px | 所有多列grid变单列 ✅ |
| padding:20px 16px!important | ≤640px | 减小内边距 ✅ |
| .desktop-nav → display:none | ≤768px | 隐藏桌面导航 ✅ |
| .mobile-menu-btn → 显示 | ≤768px | 显示移动端菜单按钮 ✅ |
| button/a → min-height/width:44px | ≤768px | Touch目标达标 ✅ |
| font-size:12px → 14px!important | ≤768px | 强制最小字体 ✅ |
| flex-wrap:wrap | ≤640px | flex容器自动换行 ✅ |
| flex-direction:column | ≤640px | 横向flex变纵向 ✅ |
| table → overflow-x:auto | 全局 | 表格可横滑 ✅ |
| 4/5列grid → 2列 | 641-1024px | iPad适配 ✅ |
