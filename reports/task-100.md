```

### 1.2 品牌Slogan
```
中文：连接专家与研究者，让知识高效流动
英文：Connecting Experts and Researchers, Making Knowledge Flow Efficiently
```

### 1.3 品牌使命
```

```

### 1.4 品牌愿景
```
成为中国专家对接平台的领跑者，连接10000+产业专家与全球顶尖研究机构。
```

### 1.5 品牌价值观
```
1. 专业（Professional）

2. 客观（Objective）
   • 提供独立、客观、真实的专家见解
   • 不偏不倚，不夸大、不隐瞒

3. 合规（Compliant）
   • 严格遵守法律法规
   • 保护用户隐私，维护数据安全

4. 高效（Efficient）
   • AI智能匹配，2小时找到专家
   • 持续优化，提升效率

5. 共赢（Win-Win）
   • 专家获得收入和能力提升
   • 研究员获得高质量专家见解
   • 平台获得成长和口碑

## 二、品牌声音

### 2.1 品牌语调
```
✅ 应该这样表达
• 专业但不晦涩（用通俗语言解释专业概念）
• 自信但不傲慢（展示实力，但不吹嘘）
• 温暖但不煽情（关心用户，但不情绪化）
• 简洁但不简陋（信息完整，但不冗长）

❌ 不应该这样表达
• 过于技术化（堆砌专业术语）
• 过于销售化（硬推销、夸大宣传）
• 过于随意（不专业、不严谨）
• 过于冗长（信息过载、重点不突出）
```

### 2.2 品牌词汇表
```
✅ 推荐使用
• 专家对接平台（不是"中介"）
• 研究员（不是"客户"）
• 专家（不是"顾问"）
• 咨询（不是"聊天"）
• 匹配（不是"介绍"）

❌ 避免使用
• 中介（暗示"中间商赚差价"）
• 客户（暗示"交易关系"）
• 顾问（暗示"长期雇佣"）
• 聊天（暗示"不专业"）
• 介绍（暗示"简单匹配"）
```

## 三、视觉识别

### 3.1 Logo使用规范
```
**Logo版本**
• 完整版（图标+中文+英文）：用于官网首页、品牌手册
• 简化版（图标+中文）：用于App、社交媒体
• 图标版：用于Favicon、App图标

**最小尺寸**
• 完整版：宽度≥200px
• 简化版：宽度≥150px
• 图标版：宽度≥50px

**安全距离**
• Logo周围保留空白区域（≥Logo高度的1/2）

**禁止行为**
• 不得拉伸、压缩、旋转
• 不得修改颜色（除非灰度版）
• 不得添加特效（阴影、发光、渐变）
```

---

### 3.2 色彩体系
```
**主色**
• ProLink Blue: #007bff
  - 象征：专业、信任、稳重
  - 使用场景：Logo、标题、按钮、链接

**辅色**
• Success Green: #28a745
  - 象征：成功、完成、积极
  - 使用场景：成功提示、完成状态

• Warning Yellow: #ffc107
  - 象征：警告、注意、待处理
  - 使用场景：警告提示、待处理状态

• Danger Red: #dc3545
  - 象征：危险、错误、紧急
  - 使用场景：错误提示、删除按钮

**中性色**
• Dark: #343a40（文字）
• Gray: #6c757d（次要文字）
• Light: #f8f9fa（背景）
• White: #ffffff（背景）
```

### 3.3 字体系统
```
**中文**
• 标题：思源黑体 Bold（Source Han Sans Bold）

• 备用：PingFang SC、Microsoft YaHei

**英文**
• 标题：Inter Bold
• 正文：Inter Regular
• 备用：Arial、Helvetica

**字号**
• H1: 32px
• H2: 24px
• H3: 20px
• 正文: 16px
• 小字: 14px
• 最小: 12px（不得更小）
```

### 3.4 图片风格
```
**专家照片**
• 风格：专业、自然、真实
• 背景：纯色或简约办公环境
• 表情：微笑、自信
• 禁止：过度修图、虚假背景

**产品截图**
• 风格：简洁、清晰、高保真
• 分辨率：2x（Retina）
• 边框：1px solid #dee2e6

**插图**
• 风格：扁平化、线性、现代
• 色彩：使用品牌色系
• 主题：专家对接、知识流动、技术感
```
