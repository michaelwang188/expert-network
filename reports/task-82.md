# 任务#82：平台FAQ文案库（专家版+研究员版各15条）

## 专家版FAQ（15条）

### 1. 关于平台
**Q：产研通 ProLink是什么？**
A：产研通 ProLink是一个合规优先的产业专家调研对接平台，帮助研究员快速找到合适的专家，完成专业调研。我们为中国领先的研究机构、咨询公司、投资机构提供专家对接服务。

**Q：如何成为产研通的专家？**
A：注册 → 填写专家资料 → 等待审核（1-2个工作日） → 审核通过后即可接单。

**Q：成为专家需要收费吗？**
A：完全免费。我们不收取任何入驻费用。

### 2. 关于接单
**Q：如何接单？**
A：研究员会主动预约您，或者您可以在"可预约时间"中等待研究员预约。预约成功后，您会收到邮件和站内信通知。

**Q：接单有强制要求吗？**
A：没有。您可以选择接受或拒绝预约（但建议您提前设置好可预约时间，避免冲突）。

**Q：如何设置可预约时间？**
A：进入"专家中心" → "可预约时间" → 设置您可用的时间段。研究员只能在这些时间段内预约您。

### 3. 关于积分
**Q：积分是什么？如何计算？**
A：1积分 = 1元人民币。您的积分费率由您自己设定（平台有建议费率范围）。

**Q：如何获得积分？**
A：完成调研后，研究员会为您的服务评分。评分≥4星，积分自动到账。

**Q：积分可以提现吗？**
A：可以。最低提现金额1000积分（即1000元），提现手续费2%，到账时间1-3个工作日。

**Q：积分有有效期吗？**
A：有，积分到账后1年内有效，到期未提现的积分将自动作废。

### 4. 关于评价
**Q：研究员给我差评怎么办？**
A：如果您认为评价不公正，可以联系平台客服申诉。平台会人工审核。

**Q：评价会影响我的排名吗？**
A：会。评分越高，被研究员选择的概率越大。评分<3星的专家会被下架。

### 5. 关于合规
**Q：调研过程中可以签署NDA吗？**
A：可以。平台支持研究员和专家之间签署NDA（保密协议）。

**Q：哪些话题不能讨论？**
A：涉及国家安全、军事机密、公司内部敏感信息的话题不能讨论。请遵守平台《合规指引》。

**Q：可以私下联系研究员吗？**
A：不建议。所有沟通应通过平台内置聊天工具进行，以便平台保障双方权益。

## 研究员版FAQ（15条）

### 1. 关于平台
**Q：产研通 ProLink是什么？**
A：产研通 ProLink是一个合规优先的产业专家调研对接平台，帮助研究员快速找到合适的专家，完成专业调研。我们为中国领先的研究机构、咨询公司、投资机构提供专家对接服务。

**Q：平台有哪些专家？**
A：我们的专家库涵盖金融科技、医疗健康、人工智能、新能源、消费零售等多个领域，所有专家都经过平台审核。

**Q：如何发布调研需求？**
A：进入"发布需求"页面 → 填写调研主题、领域、专家要求、预算等 → 提交需求，平台会在24小时内为您推荐3-5位符合条件的专家。

### 2. 关于选择专家
**Q：如何选择合适的专家？**
A：建议您综合考虑以下因素：
- 评分和订单数：评分≥4.5星且完成订单数≥10个的专家，通常服务质量较稳定
- 专业领域匹配度：选择与您的调研需求专业领域完全匹配的专家
- 可预约时间：选择可预约时间与您的需求时间匹配的专家
- 积分费率：结合评分和经验综合判断

**Q：可以查看专家的历史评价吗？**
A：可以。在专家详情页，您可以查看其他研究员对这位专家的历史评价。

### 3. 关于积分
**Q：积分是什么？如何充值？**
A：1积分 = 1元人民币。您可以通过支付宝/微信支付充值积分。充值档次：1000积分、5000积分（赠500）、10000积分（赠1200）。

**Q：如何支付专家费用？**
A：下单时，系统会根据专家费率和下单时长计算所需积分，并从您的账户锁定相应积分。调研完成后，您为专家评分≥4星，积分自动支付给专家。

**Q：可以退款吗？**
A：可以。如果专家未按时完成调研，或者调研质量严重不符合要求，可以申请退款。

**Q：积分有有效期吗？**
A：有，充值积分有效期为1年，到期未使用的积分将自动作废。

### 4. 关于调研
**Q：如何与专家沟通？**
A：通过平台内置聊天工具与专家沟通调研细节。请遵守合规要求，不得泄露敏感信息。

**Q：调研完成后如何评价专家？**
A：进入"我的订单"页面 → 找到已完成订单 → 点击"评价专家" → 评分并写下评价。

**Q：评价会影响专家的排名吗？**
A：会。评分越高，专家越容易被其他研究员选择。

### 5. 关于合规
**Q：可以签署NDA吗？**
A：可以。平台支持研究员和专家之间签署NDA（保密协议）。

**Q：哪些话题不能讨论？**
A：涉及国家安全、军事机密、公司内部敏感信息的话题不能讨论。请遵守平台《合规指引》。

**Q：可以私下联系专家吗？**
A：不建议。所有沟通应通过平台内置聊天工具进行，以便平台保障双方权益。
