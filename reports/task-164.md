## 🔐 二、登录与权限

### 2.1 登录流程

1. 访问后台登录地址
2. 输入管理员账号和密码
3. 输入二次验证码（如果已开启）
4. 点击"登录"

**注意事项**：
- ⚠️ 管理员账号需由超级管理员创建
- ⚠️ 连续5次登录失败，账号将被锁定30分钟
- ⚠️ 建议定期修改密码（每3个月）

### 2.2 权限等级

| 权限等级 | 说明 | 可执行操作 |
|----------|------|------------|
| **超级管理员** | 最高权限 | 所有操作 |
| **运营管理员** | 中等权限 | 专家管理、企业用户管理、订单管理、内容管理 |
| **客服人员** | 基础权限 | 查看订单、处理用户咨询 |
| **审核人员** | 专项权限 | 专家审核、内容审核 |

### 2.3 权限管理

**路径**：系统设置 → 管理员管理 → 权限设置

**操作**：
1. 点击"新增管理员"
2. 填写管理员信息（姓名、邮箱、手机号）
3. 选择权限等级
4. 发送邀请邮件
5. 管理员收到邮件后，点击链接设置密码

## 👥 三、专家管理

### 3.1 专家列表

**路径**：专家管理 → 专家列表

**功能**：
- 查看所有专家（已认证、待审核、已拒绝）
- 搜索专家（按姓名、领域、地区）
- 筛选专家（按认证状态、等级、注册时间）

**列表字段**：
| 字段 | 说明 |
|------|------|
| 专家姓名 | 真实姓名或化名 |
| 领域 | 一级+二级行业 |
| 认证状态 | 已认证/待审核/已拒绝 |
| 认证等级 | 顶级/资深/认证/初级 |
| 注册时间 | 加入平台时间 |
| 最近登录 | 最近一次登录时间 |
| 操作 | 查看详情/编辑/禁用 |

### 3.2 专家审核

**路径**：专家管理 → 专家审核

**审核流程**：
1. 查看待审核专家列表
2. 点击"查看详情"
3. 审核专家资料（学历、工作经历、成果）
4. 进行专业测试（笔试+面试）
5. 做出审核决定：
   - ✅ **通过**：专家获得"已认证"徽章
   - ❌ **拒绝**：填写拒绝原因，通知专家
   - 🔄 **补充材料**：通知专家补充材料

**审核标准**：
- 学历真实性（学信网验证）
- 工作经历真实性（LinkedIn+电话核实）
- 专业能力（笔试+面试）

**注意事项**：
- ⚠️ 审核需在3个工作日内完成
- ⚠️ 拒绝需填写详细原因
- ⚠️ 如不确定，可请求上级审核

### 3.3 专家禁用/启用

**路径**：专家管理 → 专家列表 → 操作 → 禁用/启用

**禁用条件**：
- 专家提供虚假材料
- 专家被企业投诉（核实属实）
- 专家违反平台规则

**禁用后果**：
- 专家无法登录
- 专家主页显示"已禁用"
- 正在进行中的订单需妥善处理

**启用条件**：
- 专家申诉成功
- 违规行为已纠正

## 📦 五、订单管理

### 5.1 订单列表

**路径**：订单管理 → 订单列表

**订单状态**：
| 状态 | 说明 | 操作 |
|------|------|------|
| 待支付 | 企业已下单，未支付 | 提醒支付 |
| 已支付 | 企业已支付，待专家确认 | 无 |
| 进行中 | 专家已确认，咨询进行中 | 查看进度 |
| 已完成 | 咨询已完成，待评价 | 提醒评价 |
| 已取消 | 订单已取消 | 查看原因 |
| 已退款 | 订单已退款 | 无 |

### 5.2 订单详情

**路径**：订单管理 → 订单列表 → 操作 → 查看详情

**详情内容**：
- 订单基本信息（订单号、创建时间、状态）
- 企业信息（公司名称、联系人）
- 专家信息（姓名、领域、等级）
- 咨询内容（需求描述、咨询时长、咨询方式）
- 支付信息（金额、支付方式、支付时间）
- 操作记录（状态变更记录）

### 5.3 订单干预

**路径**：订单管理 → 订单列表 → 操作 → 干预

**可干预操作**：
- **修改订单状态**：如企业将"进行中"误操作为"已完成"，可修正
- **退款**：如专家未完成咨询，可强制退款
- **重新匹配专家**：如专家无法完成咨询，可为企业重新匹配专家

**注意事项**：
- ⚠️ 订单干预需填写原因
- ⚠️ 干预操作会被记录到操作日志
- ⚠️ 如非必要，不建议干预订单

## 📝 六、内容管理

### 6.1 报告管理

**路径**：内容管理 → 报告管理

**功能**：
- 发布行业报告
- 编辑报告
- 下架报告

**发布流程**：
1. 点击"新增报告"
2. 填写报告信息（标题、摘要、正文、封面图）
3. 设置发布时间（立即发布或定时发布）
4. 提交审核
5. 审核通过后，报告发布

### 6.2 案例管理

**路径**：内容管理 → 案例管理

**功能**：
- 发布成功案例
- 编辑案例
- 下架案例

**发布流程**：与报告管理类似

### 6.3 内容审核

**路径**：内容管理 → 内容审核

**审核内容**：
- 专家发布的内容（文章、回答）
- 企业发布的内容（评价、需求描述）

**审核标准**：
- 无敏感词
- 无虚假信息
- 无侵权内容

## ❓ 九、常见问题FAQ

### 9.1 登录相关

**Q1：忘记管理员密码怎么办？**
A：点击登录页面的"忘记密码"，输入邮箱，重置密码链接将发送到邮箱。

**Q2：账号被锁定怎么办？**
A：等待30分钟自动解锁，或联系超级管理员解锁。

### 9.2 专家管理相关

**Q3：专家审核需要多长时间？**
A：一般情况下，3个工作日内完成审核。

**Q4：专家被误禁用怎么办？**
A：进入专家详情页，点击"启用"，填写启用原因。

### 9.3 订单管理相关

**Q5：企业投诉专家怎么办？**
A：进入订单详情页，查看投诉内容，核实后做出处理（警告专家/退款/禁用专家）。

**Q6：订单状态异常怎么办？**
A：进入订单详情页，点击"干预"，手动修正订单状态。

### 9.4 其他问题

**Q7：如何导出数据报表？**
A：进入数据报表页面，选择时间范围和指标，点击"导出Excel"。

**Q8：如何查看操作日志？**
A：进入系统设置 → 操作日志，可查看所有操作记录。
