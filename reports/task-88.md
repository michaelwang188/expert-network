## 一、推送通知文案库总体框架

### 文案定位
面向产研通ProLink App用户的推送通知文案库，覆盖订单、积分、系统3大类共30条通知，确保用户及时获取关键信息，提升活跃度和留存率。

### 文案原则
1. **简洁明了** - 20字以内，一眼看懂
2. **行动导向** - 明确告知用户要做什么
3. **个性化** - 包含用户姓名、订单号等信息
4. **时效性** - 紧急事项立即通知，非紧急可批量
5. **友好温暖** - 不用"必须""强制"等强硬词汇

## 五、推送通知最佳实践

### 5.1 推送时间

**最佳推送时间**
```
✅ 工作日：9:00-11:00，14:00-17:00，19:00-21:00
✅ 周末：10:00-12:00，15:00-17:00，20:00-22:00

❌ 避免：22:00-8:00（除非紧急事项）
```

**紧急事项立即推送**
```
• 订单确认（专家接受/拒绝）
• 咨询即将开始（提前1小时）
• 账户安全（异地登录、密码修改）
```

**非紧急事项批量推送**
```
• 积分到账（可延迟1分钟，合并多条）
• 系统更新（可定时推送，如周五下午）
• 活动推广（可选时间推送）
```

---

### 5.2 推送频率

**推送频率限制**
```
• 同一用户：每天最多5条（不含紧急事项）
• 同一类型：每小时最多1条
• 活动推广：每周最多2条

如用户投诉"推送太频繁"，自动降低频率或让用户自定义
```

## 七、推送通知数据指标

### 7.1 核心指标

**推送效果指标**
```
• 发送量：成功发送的推送数量
• 到达率：成功到达用户设备的比例（目标>95%）
• 打开率：用户打开推送的比例（目标>30%）
• 点击率：用户点击推送的比例（目标>10%）
• 转化率：用户完成目标行动的比例（目标>5%）
```

**用户满意度指标**
```
• 投诉率：用户投诉推送的频率（目标<1%）
• 退订率：用户关闭推送的比例（目标<5%）
• NPS：用户推荐平台的比例（目标>50）
```
