### 1.6 费用说明

**标题**: 费用说明

**费用构成**
```
订单金额 = 专家服务费 + 平台服务费（已含）

专家服务费：支付给专家的费用
平台服务费：平台收取的费用（30%，由专家承担）

研究员实际支付 = 专家服务费
专家实际收入 = 专家服务费 × 70%
```

**支付方式**
```
• 支付宝
• 微信支付
• 银行转账（对公账户）
• 信用卡（Visa、MasterCard、银联）
```

**发票开具**
```
• 电子发票（增值税普通发票）
• 纸质发票（增值税专用发票，企业客户）
• 发票内容：咨询服务费
• 开票时间：订单完成后7个工作日内
```

**退款政策**
```
• 专家未确认：全额退款（3-7个工作日）
• 专家已确认，咨询前24小时以上：扣除¥100手续费，剩余退款（3-7个工作日）
• 专家已确认，咨询前24小时内：扣除50%费用，剩余退款（3-7个工作日）
• 咨询已完成：不支持退款（除非专家未参加或质量差，可申诉）
```

### 1.7 FAQ

**标题**: 定价常见问题

**Q1：价格可以议价吗？**
A：不可以。平台采用统一定价，确保公平性。但您可以选择不同等级的专家（费率不同），或购买会员享受折扣。

**Q2：会员费可以退款吗？**
A：不可以。会员费不支持退款，但您可以在会员到期前取消自动续费。

**Q3：批量优惠可以和其他优惠叠加吗？**
A：不可以。批量优惠不可与会员折扣叠加，系统会自动选择最优惠的价格。

**Q4：我可以邀请专家入驻吗？**
A：可以。邀请专家入驻并获得认证，您可获得200积分奖励，专家可获得费率优惠（前3个月8折）。

**Q5：发票可以开增值税专用发票吗？**
A：可以。企业客户可申请增值税专用发票，需提供税务登记证、开户许可证。
