## 五、法律审核要点

### 5.1 需要法务重点审核的条款
```
1. 争议解决条款
   - 仲裁 vs 诉讼
   - 管辖法院选择

2. 免责条款
   - 不可抗力定义
   - 免责范围

3. 违约责任条款
   - 赔偿范围
   - 赔偿上限

4. 隐私条款
   - 是否符合《个人信息保护法》
   - 用户权利是否完整

5. 知识产权条款
   - 平台IP归属
   - 用户IP归属
   - 侵权责任
```

---

### 5.2 法律风险点
```
1. 跨境数据传输
   - 如有境外用户，需符合数据出境安全评估

2. 格式条款
   - 服务协议、隐私政策属于格式条款
   - 需显著提示（加粗、弹窗）
   - 不得排除用户主要权利

3. 未成年人保护
   - 需添加未成年人保护条款
   - 不得向未成年人提供专家对接服务

4. 行业特殊规定
   - 如涉及医疗、金融，需符合行业规定
```
