## 四、违规处理机制

### 4.1 扣分规则
```
## 扣分累计制

**扣分标准**
• 基础分：100分
• 严重违规：直接封号（0分）
• 一般违规：按情节扣5-20分

**扣分后果**
• 90-99分：警告通知
• 80-89分：限制接单（7天）
• 70-79分：降级处理（推荐专家→普通专家）
• 60-69分：限制接单（30天）
• <60分：终止合作

**加分恢复**
• 连续30天无违规：+5分
• 获得客户好评：+2分/次
• 参加平台培训：+5分/次
• 年度优秀专家：+20分
```

### 4.2 申诉流程
```
## 申诉机制

**申诉条件**
• 对违规认定有异议
• 对扣分/处罚有异议

**申诉流程**
1. 收到处罚通知7天内提交申诉
2. 填写申诉表（说明+证据）
3. 平台5个工作日内审核
4. 审核结果通知专家
5. 如不满意，可二次申诉（最终裁决）

**申诉渠道**
• 平台内申诉入口：/expert/appeal
```

## 六、专业操守指引

### 6.1 咨询准备
```
## 咨询前准备清单

**✅ 必做事项**
1. 仔细阅读客户需求（领域、问题、预期）
2. 确认自己专业匹配度（不匹配主动拒绝）
3. 准备相关资料（数据、案例、图表）
4. 测试设备（网络、音频、视频）
5. 选择安静环境（无干扰、保隐私）

**❌ 禁止事项**
1. 不了解领域硬接单
2. 酒后、疲劳状态接单
3. 公共场所接单（咖啡厅、高铁等）
4. 边做其他事边咨询
5. 承诺无法兑现的事项
```

### 6.2 咨询进行中
```
## 咨询中行为准则

**沟通技巧**
• 语速适中、表达清晰
• 多用数据、案例、图表
• 承认不确定性（"根据我的经验..."）
• 引导客户提问（"您还想了解什么？"）

**时间管理**
• 准时开始、准时结束
• 如需要延长，征得客户同意
• 高效沟通，避免闲聊

**边界意识**
• 不讨论政治、宗教、敏感话题
• 不评价客户公司/同行（客观即可）
• 不推销自己的产品/服务
• 不索要客户联系方式
```

### 6.3 咨询结束后
```
## 咨询后跟进

**及时确认**
• 24小时内确认完成
• 填写咨询小结（关键词、要点）
• 上传相关资料（如承诺）

**客户反馈**
• 礼貌请求客户评价
• 对负面评价理性回应
• 不争论、不反驳、不情绪化

**持续改进**
• 查看客户评价，找出改进点
• 参加平台培训，提升能力
• 更新个人档案，补充新案例
```

## 七、合规红线

### 7.1 信息披露
```
## 什么可以说，什么不能说

**✅ 可以说**
• 公开信息（财报、新闻、行业报告）
• 个人观点（明确标注"个人观点"）
• 行业趋势（基于公开数据的分析）
• 方法论（通用分析方法、框架）

**❌ 不能说**
• 客户商业秘密（未公开的产品、战略、数据）
• 内幕信息（未公开的并购、融资、业绩）
• 他人隐私（同事、客户、第三方的私密信息）
• 敏感信息（政治、军事、外交）
```

### 7.2 利益冲突
```
## 识别与申报利益冲突

**常见利益冲突场景**
• 您是客户竞争对手的员工
• 您持有客户股票/衍生品
• 您与客户有业务往来
• 您的亲属在客户公司任职

**申报流程**
1. 发现冲突→立即申报（/expert/conflict）
2. 平台评估→决定是否继续
3. 如继续→签署保密协议+利益冲突声明
4. 如拒绝→平台补偿（¥200咨询费）

**未申报后果**
• 首次发现：警告+扣分20
• 再次发现：降级+罚款¥1000
• 造成严重后果：封号+追责
```

### 8.1 客户隐私保护
```
## 保护客户隐私，人人有责

**您必须遵守**

• 端到端加密（AES-256）

• 审计日志（所有操作可追溯）
```

### 8.2 专家隐私保护
```
## 您的隐私，平台守护

**匿名模式**

• 真实姓名和机构在订单确认后可见
• 可设置"仅展示领域，不展示细分"

**信息保护**
• 联系方式仅用于平台通知
• 不向客户、第三方泄露专家联系方式

**数据安全**
```

## 十一、FAQ

### 11.1 常见疑问
```
## 专家最常问的10个问题

**Q1：我可以拒绝订单吗？**
A：可以。您有权自主选择接受或拒绝订单，但频繁拒绝（拒绝率>50%）会影响您的评级和派单优先级。

**Q2：客户问我股票代码，我可以说吗？**
A：不可以。这属于内幕信息或投资建议，违反合规红线。您可以分享行业趋势、竞争格局等公开信息。

**Q3：咨询结束后，客户加我微信，我可以通过吗？**
A：不可以。这属于"绕过平台私下交易"，违反平台规则。请引导客户通过平台联系您。

**Q4：我对客户评价有异议，怎么办？**
A：您可以在7天内提交申诉（/expert/appeal），平台会人工审核。如评价不实，平台会删除并恢复您的评分。

**Q5：我的费率可以自己定吗？**
A：基础专家费率由平台建议（¥800-2000/小时），推荐专家可申请上调（最高¥5000/小时）。最终费率需平台审核。

**Q6：我可以同时接多个平台的单吗？**
A：可以。但需确保不泄露客户信息、不利用ProLink的客户资源为其他平台服务。

**Q7：咨询中客户问的问题超出我的领域，怎么办？**
A：诚实告知"这个问题超出我的专业范围，我可以为您推荐其他专家"。不要硬答，以免误导客户。

**Q8：我的收入需要报税吗？**
A：需要。平台会为您代扣代缴个人所得税（20%），并提供完税凭证。年度汇总收入需您自行申报汇算清缴。

**Q9：我被客户投诉了，怎么办？**
A：保持冷静，联系平台客服了解投诉内容。如您无责，提交证据申诉；如您有责，道歉并改进，避免再次犯错。

**Q10：我想退出平台，怎么办？**
A：可以在/Expert/settings申请退出。平台会在30天内结算所有未结费用。如您有进行中的订单，需完成后再退出。
```
