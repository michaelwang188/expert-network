## 资金安全保障文案

### 保障首页（服务保障页顶部）

```markdown
# 💰 资金安全保障

您的每一笔积分交易，都由 ProLink 三重安全机制守护。

---

## 🔐 三重安全保障

### 第一重：积分闭环管理
🛡️ 积分仅在平台内流转，不可提现、不可转让、不可私下交易
🛡️ 从充值到结算，每一笔积分都有全链路可追溯记录
🛡️ 积分绑定您的实名账户，杜绝他人冒用

### 第二重：交易原子保护
⚛️ 订单结算采用数据库事务机制，保证积分不丢、不超扣
⚛️ 研究员扣分和专家加分在同一事务中完成——要么全成功，要么全回滚
⚛️ 并发安全设计：即使多人同时操作，也不会出现积分错误

### 第三重：全程可追溯
📋 每一笔积分变动都有明确记录（时间、类型、金额、关联订单）
📋 您可随时在"我的积分"中查看完整流水
📋 平台保留交易记录至少 5 年，支持争议核查

## 积分规则

| 项目 | 规则 | 说明 |
|------|------|------|
| 积分有效期 | 永久有效 | 不设过期时间 |
| 积分转让 | 不支持 | 仅限本人账户使用 |
| 积分提现 | 不支持 | 平台内结算专用 |
| 积分清零 | 不会无故清零 | 仅违规封禁时依法处置 |
| 充值比例 | 1元 = 10积分 | 以实际活动为准 |

---

## 角色积分规则

### 研究员（消费者）
- 充值获得积分 → 下订单消费积分 → 访谈完成确认结算
- 未消费的积分长期有效
- 因平台或专家原因取消的订单，积分自动退还

### 专家（提供者）
- 完成访谈获得积分 → 积累劳动报酬
- 积分不可兑现金（合规隔离设计）
- 违规操作（如私下交易）将冻结结算

---

```
研究员充值积分  →  创建订单（冻结积分）  →  专家接单
                                            ↓

                                            ↓
                           ┌────────────────┴────────────────┐
                           ↓                                 ↓
                    研究员账户扣减积分                  专家账户增加积分
                           └────────────────┬────────────────┘
                                            ↓
                                    结算完成，双流水记录

```markdown
# 📋 退款与取消政策

---

## 退款条件

以下情况，积分将自动退还至研究员账户：

| 情况 | 退款规则 | 退款时间 |
|------|---------|---------|
| 订单超时未被专家确认 | 全额退还 | 超时后即时 |
| 专家主动取消订单 | 全额退还 | 取消后即时 |
| 平台判定订单无效 | 全额退还 | 判定后1个工作日内 |
| 访谈未能正常进行（平台责任）| 全额退还 | 确认后1个工作日内 |
| 研究员下单后1小时内取消 | 全额退还 | 取消后即时 |

---

## 不可退款情况

以下情况，积分不予退还：

| 情况 | 原因 |
|------|------|
| 访谈正常完成，研究员确认满意 | 服务已完成 |
| 访谈完成后7日内未提出异议 | 视为确认满意，自动结算 |
| 因研究员原因未能按时参加访谈 | 专家时间已预留 |
| 访谈已进行超过一半后研究员取消 | 服务已实质性提供 |
| 研究员违反合规协议导致访谈中止 | 研究员违约在先 |

---

## 退款流程

### 自动退款（无需操作）
```
研究员取消订单 → 系统自动退还积分 → 积分即时到账
```

### 争议退款（需平台介入）
```
研究员发起争议    →   平台审核（1-3个工作日）
                            ↓
                 ┌─────────┴─────────┐
                 ↓                   ↓
            争议成立              争议不成立
          积分退还研究员          积分划转专家
                 ↓                   ↓
            结果通知双方          结果通知双方
```

---

## 争议处理标准

### 争议成立（退还研究员）的情况：

1. **专家未按时出现**：超过约定时间30分钟未上线
2. **专家身份不符**：实际参与访谈者与预约专家不一致
3. **专家拒绝合规录音**：无正当理由拒绝平台录音
4. **专家中途终止访谈**：非因研究员原因中途退出
5. **合规违规中止**：因专家违规导致访谈被平台中止

### 争议不成立（归专家）的情况：

1. **研究员主观不满**：访谈完成但因"收获不大"要求退款
2. **研究员迟到导致时间不足**：不是专家的问题
3. **研究员不配合录音**：因研究员拒绝录音导致访谈无法进行
4. **服务已实质性完成**：访谈已进行超过80%
5. **超时未提争议**：完成后超过7天未提出
```
