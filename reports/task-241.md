## 设计原则

### 文案调性
- **专业权威**：展示专业能力和行业经验
- **简洁有力**：用最少的文字传达最大价值
- **真实可信**：避免夸大，用具体数据和案例说话
- **用户导向**：从研究员需求角度出发

### 格式规范
- 擅长话题：200-500字（精炼但充分）
- 技能标签：5-10个（精准匹配需求）
- 访谈形式：明确标注（线上/线下/混合）
- 所有文案使用中文，专业术语可保留英文

## 文案撰写指南

### 1. 头衔（title）撰写指南

**作用**：第一眼展示专家身份和专业方向

**格式**：`{当前职位} · {核心专业领域}`

**示例**：
- ✅ "首席技术官 · 人工智能与大数据"
- ✅ "研究总监 · 新能源汽车产业链"
- ✅ "合伙人 · 医疗健康投资"
- ❌ "专家"（太模糊）
- ❌ "CTO"（不够具体）

**行业映射表**：

| 行业 | 推荐头衔格式 | 示例 |
|------|------------|------|
| 科技互联网 | {职位} · {技术领域} | 技术VP · 云计算架构 |
| 金融 | {职位} · {金融领域} | 首席分析师 · 资产管理 |

### 4. 擅长话题（topics）撰写指南

**作用**：核心介绍文案，展示专家能力和经验

**格式规范**：
- 长度：200-500字
- 结构：能力概述 → 具体经验 → 可提供的价值
- 语调：专业、自信、具体

**模板**：

```markdown
【能力概述】（1-2句）
我是{机构}{职位}，拥有{X}年{行业/领域}经验。

【具体经验】（2-3句）
曾主导/参与{具体项目/业务}，涵盖{关键领域1}、{关键领域2}、{关键领域3}。
熟悉{行业/领域}的{痛点1}、{痛点2}、{痛点3}。

【可提供的价值】（2-3句）
可以提供关于{话题1}、{话题2}、{话题3}的深入洞察。
对于{具体问题}，可以提供实操建议和案例分析。
```

**示例文案**：

**示例1：科技互联网 - AI方向**
```
我是腾讯首席科学家，拥有15年人工智能研发经验。

曾主导腾讯大模型研发，涵盖自然语言处理、多模态理解、模型压缩等核心技术。
熟悉AI在搜索、推荐、广告、内容理解等场景的落地应用，以及AI产品的商业化路径。

可以提供关于AI技术趋势、大模型应用场景、AI产品化策略的深入洞察。
对于AI团队的搭建和技术选型，可以提供实操建议和案例分析。
```

**示例2：金融 - 资管方向**
```
我是易方达基金投资总监，拥有12年资产管理经验。

曾管理超过500亿规模的基金产品，涵盖股票、债券、衍生品等多资产类别。
熟悉中国资本市场的监管环境、投资策略、风险控制体系，以及ESG投资的最新趋势。

可以提供关于资产管理行业趋势、投资策略设计、风险控制的深入洞察。
对于资管产品的设计和运营，可以提供实操建议和案例分析。
```

**示例3：医疗健康 - 医药方向**
```
我是恒瑞医药医学总监，拥有10年新药研发和临床试验经验。

曾主导5个创新药的临床试验，涵盖肿瘤、免疫、代谢等疾病领域。
熟悉新药研发全流程，从药物发现、临床前研究到临床试验、注册申报。

可以提供关于新药研发趋势、临床试验设计、注册申报策略的深入洞察。
对于医药企业的研发管线规划，可以提供实操建议和案例分析。
```

## 文案优化建议

### 优化点1：头衔优化

**问题**：头衔过于模糊，无法快速判断专家方向

**优化前**：`专家`、`顾问`、`总监`

**优化后**：`首席技术官 · 人工智能与大数据`、`独立顾问（前麦肯锡）· 战略规划`、`市场总监 · 快消品数字化`

**工具**：提供头衔生成器，根据职位和行业自动生成推荐头衔

---

### 优化点2：擅长话题优化

**问题**：话题过于笼统，缺乏具体价值

**优化前**：
```
我有丰富的行业经验，可以提供专业的建议。
```

**优化后**：
```
我拥有X年{行业/领域}经验，曾主导/参与{具体项目/业务}。
可以提供关于{话题1}、{话题2}、{话题3}的深入洞察。
对于{具体问题}，可以提供实操建议和案例分析。
```

**工具**：提供话题模板生成器，引导专家按照模板填写

---

### 优化点3：技能标签优化

**问题**：标签过于宽泛，匹配不精准

**优化前**：`管理`、`技术`、`市场`

**优化后**：`人工智能`, `机器学习`, `深度学习`, `自然语言处理`, `计算机视觉`, `大模型`

**工具**：提供标签推荐系统，根据头衔和擅长话题自动推荐标签

### 专家个人主页文案自查清单

- [ ] 头衔是否具体明确？（不是"专家"、"顾问"）

- [ ] 一级行业是否选择准确？
- [ ] 二级子领域是否选择准确？
- [ ] 三级岗位类型是否选择准确？
- [ ] 技能标签是否精准？（5-10个）

- [ ] 是否在200-500字之间？
