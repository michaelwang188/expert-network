#### 发现的问题

1. **多个页面缺少登录鉴权**
   - 严重程度：🔴致命
   - 描述：/experts、/request、/orders 三个页面未做登录鉴权，未登录用户可直接访问
   - 复现步骤：清除 cookie → 直接访问 https://expert-network-sooty.vercel.app/request → 页面正常加载，可填写并提交调研需求
   - 建议修复：在 middleware 或页面级添加 auth guard，未登录时重定向到 /login

2. **未登录访问 /orders 时 Application Error 崩溃**
   - 严重程度：🔴致命
   - 描述：未登录状态访问 /orders 页面白屏，显示 "Application error: a client-side exception has occurred"
   - 复现步骤：清除 cookie → 访问 https://expert-network-sooty.vercel.app/orders
   - 建议修复：添加鉴权守卫 + 对无 session 状态做空值保护

3. **专家端无资料编辑功能**
   - 严重程度：🔴致命
   - 描述：专家登录后导航栏仅有「数据看板」和「订单管理」，无资料上传、标签设置、档期管理、报价设置入口。需求文档第二章明确要求专家应有三级标签体系、报价设置、档期设置
   - 复现步骤：expert@demo.com 登录 → 查看导航栏 → 无任何资料编辑入口
   - 建议修复：新增专家端「我的资料」页面，含标签编辑、档期/报价设置、合规自查入口

4. **需求发布无预算/报价字段**
   - 严重程度：🟡中等
   - 描述：需求发布表单仅有调研主题、行业、细分环节、访谈提纲，缺少预算、时长、报价等核心交易字段。需求文档第四章明确要求「研究员总费用=专家劳务报酬+平台服务费」
   - 复现步骤：研究员登录 → 发起调研 → 表单中无预算/时长字段
   - 建议修复：在需求表单中增加预算范围、预期时长、报价偏好等字段

5. **研究员端无个人中心**
   - 严重程度：🟡中等
   - 描述：导航栏用户名不可点击，/profile、/settings 均返回 404
   - 复现步骤：研究员登录 → 点击导航栏用户名 → 无响应；直接访问 /profile → 404
   - 建议修复：实现 /profile 页面（基本信息、历史订单、评价记录）

6. **缺少定价与收费机制展示**
   - 严重程度：🟡中等
   - 描述：全站无专家定价展示、无平台抽成说明、订单无金额列。需求文档第四章要求「专家自主定价+平台指导价区间」「平台抽成15%-30%」「平台预收全款」
   - 复现步骤：浏览全站页面，均无定价/收费相关内容
   - 建议修复：订单模型增加金额字段，专家列表展示报价区间，需求确认页展示费用明细

7. **缺少私下交易防护**
   - 严重程度：🟡中等
   - 描述：无隐私号通话、无联系方式隐藏、无平台内沟通工具、无反绕过提示。需求文档第五章明确要求「平台专属会议室，禁止私自互换联系方式，一旦发现永久拉黑」
